# GitHub Skills Map
Visualize developer skills from GitHub activity.

## Usage
1. Generate a GitHub API token using `generate_github_api_token`.
2. Fetch a user's recent commits and pull requests using `fetch_user_commits` and `fetch_user_pull_requests`.
3. Calculate skills based on commit and pull request data using `calculate_skills`.
4. Display skills in a visual representation using `display_skills`.

## Testing
Run tests using `pytest`.
